fafrotskies
===========

ringing ringing terminal

# Requirements
.NET Framework 4 or later. This also works on Mono!

# Usage
```
fafrotskies [problem file]
```

# Problem File
It's YAML file.

```yaml
name: Problem Name
description: |
  Problem Description here!
  Let's study elementary mathemathics!
flag: FLAG{kogasachan_kawaii}
cases:
  -
    problem: 1+1=?
    answer: 2
  -
    problem: 3*3=?
    answer: 9
```
